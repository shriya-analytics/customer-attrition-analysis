# Customer Attrition & Retention Analysis

## Objective

## Tools Used
- Excel (data cleaning, pivot tables, basic analysis)
- SQL (exploratory analysis and segmentation)
- Business & consulting-style analysis

## Key Questions
- Which customer segments have the highest attrition?
- Does tenure, usage, or customer value impact churn?
- What actionable levers can reduce attrition?
